# Archived UX Components

This directory contains UI/UX patterns that were removed from the live application but saved as reference for future use.

## Chat Mode (Interactive Brainstorm)

The interactive chat mode for querying expert knowledge graphs was removed from the brainstorm page. The functionality will be integrated into the Prompt Studio instead, providing a unified creative workspace with export capabilities.

**Files:**
- `chat_mode_styles.css` - CSS styles for the chat interface
- `chat_mode_html.html` - HTML structure for the chat UI
- `chat_mode_javascript.js` - JavaScript functions for chat functionality

**Why archived:**
- Consolidating chat functionality into Prompt Studio
- Avoiding duplicate interfaces
- Better export workflow from Prompt Studio to project elements

**Potential future use:**
- Reference for designing the enhanced Prompt Studio integration
- Reusable chat UI patterns
- Expert response display formatting
